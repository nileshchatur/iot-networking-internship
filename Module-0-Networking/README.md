# Module 1 — TCP Socket Programming

This module introduces basic TCP socket programming using Python.

The objective is to understand how two applications communicate using the TCP/IP protocol stack through client-server architecture.

---

# Concepts Covered

- TCP communication
- Client-server model
- IP address and port number
- Socket creation
- Binding and listening
- Connection establishment
- Sending and receiving data
- Loopback communication using localhost (127.0.0.1)
